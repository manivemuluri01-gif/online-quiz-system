ONLINE QUIZ SYSTEM WITH RESULT ANALYTICS

Project Description
This project is a web-based quiz application that allows users to answer
multiple-choice questions and automatically generates results and performance analytics.

Technologies Used
- HTML
- CSS
- JavaScript

Features
- Multiple choice quiz questions
- Automatic score calculation
- Percentage calculation
- Performance grading system
- Restart quiz option

How to Run the Project
1. Download the project files
2. Open the file "online-quiz-system.html"
3. Run it in any web browser
